# 🐺 My OSINT Toolbox (TSG Framework)
"The world is full of obvious things which nobody by any chance ever observes." - Sherlock Holmes

This repository serves as my personal collection of tools and methodologies for Digital Investigations and Penetration Testing, organized under the **TSGINT** (Technical, Social, and Geospatial Intelligence) framework.

---

## 🛠 TECHINT (Technical Intelligence)
*Objective: Gathering technical infrastructure data and digital vulnerability discovery.*

| Tool | Primary Use Case |
| :--- | :--- |
| **Google Dorking** | Advanced operators for sensitive file discovery & log leaks. |
| **Shodan / Censys** | Scanning IoT devices, servers, and identifying open ports. |
| **Wayback Machine** | Analyzing historical Web/API snapshots to find hidden endpoints. |
| **Crt.sh / DNSDumpster** | Subdomain enumeration for attack surface mapping (Recon). |

## 👥 SOCMINT (Social Media Intelligence)
*Objective: Analyzing behaviors, relationships, and digital identities.*

* **Username Hunting:** Utilizing `Sherlock` or `WhatsMyName` to track aliases across +500 platforms.
* **Identity Verifier:** `Epieos` & `Hunter.io` for email validation and linked service discovery.
* **Credential Leak Check:** Searching underground databases for leaked credentials and data breaches.
* **Telegram Bot Intel:** Leveraging OSINT bots for quick phone/ID lookup and record verification.

## 🌍 GEOINT (Geospatial Intelligence)
*Objective: Geographic positioning and visual analysis (Specialty: Sun/Shadow & Surface analysis).*

- **[Shadowmap.org](https://shadowmap.org):** Real-time shadow analysis to determine time and location of a capture.
- **[SunCalc](https://suncalc.org):** Solar angle calculations to verify window orientations and light sources.
- **PeakVisor:** Terrain and mountain range identification from distant photography.
- **Google Earth Pro:** 3D analysis of construction materials and terrain topology.
- **ExifViewer:** Metadata extraction for GPS coordinates and camera signatures.
